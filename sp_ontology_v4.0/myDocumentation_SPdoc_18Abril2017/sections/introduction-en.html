Introduction back to ToC

The document module facilitates archiving; our SP-document module is a proposal to standardize document aspects in experimental protocols. We analyzed 530 experimental protocols gathered from several sources (the protocols are available here). From this data set, we extracted a set of metadata element that are necessary and sufficient to represent document aspects in experimental protocols.
